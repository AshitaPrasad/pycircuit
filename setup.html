Installing pycircuit¶

Requirements¶

  • Python 2.4 or 2.5
  • Numpy version >= 1.04
  • Scipy
  • Sympy

Installation¶

As root:

# python setup.py install

Or in ubuntu:

$ sudo python setup.py install
